# The Leverage Gap — AI Strategy Presentation

A cinematic, single-file presentation on the gap between people who *use* AI and people who *leverage* it.

> **Tools ≠ Advantage. Leverage = Advantage.**

---

## Quick Start

No build step required. Just open `index.html` in a browser — or serve it locally to enable translation loading:

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```

Then visit `http://localhost:8080`.

---

## Files

| File | Purpose |
|---|---|
| `index.html` | Full presentation — slides, styles, and logic in one file |
| `en.json` | English translations for all slide content |
| `ar.json` | Arabic translations for all slide content |

---

## Features

- **15 slides** — narrative arc from problem to framework
- **Bilingual** — English / Arabic with smooth animated transitions
- **Dark & Light themes** — toggle via the ☀/☾ button (top right)
- **Auto-sync mode** — automatically alternates between languages on a timer
- **Keyboard navigation** — `← →` arrow keys or space bar
- **Touch support** — swipe left/right on mobile
- **Fullscreen** — bottom-left button or browser F11

---

## Controls

| Control | Action |
|---|---|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| ☀ / ☾ button | Toggle dark / light theme |
| ⚙ button | Open settings (language sync, timer) |
| ⛶ button | Toggle fullscreen |
| Nav dots | Jump to any slide |

---

## Customization

All text content lives in `en.json` and `ar.json` — edit those files to update any slide copy without touching the HTML.
